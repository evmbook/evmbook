# Mastering EVM - Book Content (v1)

Book content source for [masteringevm.com](https://masteringevm.com).

## Structure

```
content/
├── chapters/       # Main book chapters (MDX)
├── appendices/     # Reference appendices
└── meta/           # Colophon, about, changelog

```
